## FOK ONE 抢票小程序-后台管理

> 一款微信小程序抢票系统，暂支持电影抢票，后续根据需求开发其他抢票功能

## 技术栈
1. Koa2框架
2. Vue框架
3. MySQL数据库

## 目标功能
- [ ] 电影院管理
- [ ] 电影信息管理
- [ ] 电影上线、下线管理
